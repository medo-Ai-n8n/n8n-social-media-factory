# n8n Social Media Factory 🤖

Open-source n8n workflows to build AI-powered social media automations.

## What it does
- Generate posts with Claude/GPT
- Auto-reply to Instagram DMs and Comments  
- Cross-post to Instagram, Facebook, Telegram
- AI content calendar and scheduling

## Tech Stack
- n8n
- Docker  
- Meta Graph API
- Anthropic Claude API

## Stats
- 500+ monthly downloads
- 80+ active users
- Built for non-technical creators
